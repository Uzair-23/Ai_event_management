Frontend setup for AI Event Management

1. Copy `.env.example` to `.env` and fill in `VITE_API_BASE` and `VITE_SOCKET_URL`.
2. npm install
3. npm run dev

Important: Ensure the backend is running at `VITE_API_BASE` and that the backend `.env` contains `JWT_SECRET`, `MONGO_URI`, and `UNSPLASH_ACCESS_KEY`.
